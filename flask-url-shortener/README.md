# Flask URL  SHORTENER
A simple URL shortener build with **Flask**.
Enter a long URL  and get a short,easy-to-share URL that redirects to the original link.

---
## Features
- Shortens any URL into a random 6-Charectar code.
- Automatically adds `http://`  missing.
- Redirects short URLs to the original link.
- Minimal easy to use web interface

---
## Requirements
- Python 3.10+
- Flask 2.3.2 or(Compatible version)
Install dependencies using:
```bash
pip install -r requirements.txt
```
